# 3–5 Minute Demo Video Script

## 0:00–0:30 — Context
This is a Sarvam AI powered multilingual collections agent for NBFCs and banks. The target use case is early-stage EMI reminders and recovery calls for Indian borrowers.

## 0:30–1:00 — Architecture
Show README architecture diagram. Explain: borrower voice goes to Sarvam STT, Sarvam-105B handles conversation and decisioning, workflow updates CRM/reminder/escalation, Sarvam TTS replies in borrower language.

## 1:00–2:15 — Hindi/Hinglish Demo
Select borrower BRW001.
Start call.
Borrower response: “Salary abhi nahi aayi, Friday ko payment kar dunga.”
Show the agent accepting promise-to-pay.
Show CRM event and reminder queue update.

## 2:15–3:15 — Risk Escalation Demo
Type: “Ye loan mera nahi hai, galat call hai.”
Show dispute response.
Show escalation queue update.

## 3:15–4:15 — Business Value
Explain ROI assumptions: 10 lakh monthly calls, 60% automation, INR 45 human cost vs INR 6 AI cost, estimated INR 2.34 crore monthly savings.

## 4:15–5:00 — Why Sarvam
Sarvam is valuable because this is an India-specific voice problem: Indian languages, accents, code-mixing, local context, and regulated enterprise deployment.
